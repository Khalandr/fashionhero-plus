# FashionHero Plus — prototype

Smoke-test landing page for a buyer Premium membership at FashionHero (a fictional Polish fashion marketplace).

Built as quest 1.3 ("From Zero to Demo") in the [AI Product Heroes](https://questlog.aiproductheroes.pl/) course, 2nd edition. The hypothesis being tested: buyers see enough value in a bundled premium membership (free express shipping + early access + curated capsules + 30-day returns + AI Style Concierge) to leave their email at 19,90 zł/month or 199 zł/year.

**Live:** https://khalandr.github.io/fashionhero-plus/

This prototype is intentionally throwable — a smoke test for desirability, not a product.
